# sass-basic-starter
Sass ile oluşturduğum arayüzler için hızlı bir başlangıç.
Compailer:Live Sass Compiler v5.4.0("Glenn Marks")
> settings.json
```
    "liveSassCompile.settings.formats": [
        {
            "format": "expanded",
            "extensionName": ".css",
            "savePath": "/dist",
            "savePathSegmentKeys": null,
            "savePathReplaceSegmentsWith": null
        }
    ]
```
